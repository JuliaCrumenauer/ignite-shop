# Sobre

O ignite shop é um e-commerce, onde é possível navegar em um carrossel de camisetas, selecionar o produto desejado e adquirir. 

Projeto desenvolvido com Next.js e ReactJs.

# Tecnologias Utilizadas
- Keen slider
- Stripe
